Android开发一：getItemViewType越界

2019-03-25
在RecyclerView.Adapter中显示多种不同Item，如果设置的getItemViewType的返回值超过getItemCount-1值，或者getItemViewType(0)>0就会越界 

    @Override
    public int getItemViewType(int position) {
        Log.e("android", " " + position + "    " + mList.size());
        if(mList.get(position).getType()==0){
            return Type_right;
        }
        return Type_left;
    }
